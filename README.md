# Baltistan Silicon Lab Exam Portal

Professional AI/Data Science Exam Portal with automated scheduling, complex MCQ banks, and branded PDF results.

## Quick Start

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Run the project:**
    ```bash
    npm start
    ```

3.  **Open in browser:**
    [http://localhost:3000](http://localhost:3000)

## Features
- **Branding**: Official "Baltistan Silicon Lab" results.
- **Advanced MCQs**: 100+ complex questions per class.
- **Dynamic Scheduling**: Admin-controlled time slots and durations.
- **Student Dashboard**: Live visibility of upcoming and active exams.
- **PDF Results**: Professional-grade downloadable result sheets with full Q&A review.

## Tech Stack
- React.js
- React Router
- EmailJS (for automated result delivery)
- CSS3 (Print-optimized for PDF export)
